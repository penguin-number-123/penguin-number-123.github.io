Powerful arrow Notation and Chained variant.

Note that all variables in this function are natural numbers. For PAN, the domain would be \(\mathbb{N}^3\). For the chained varaiant, we would have the domain as \(\mathbb{N}^n\) for \(n \in \mathbb{N}\)
In addition, PAN never uses multiple up arrows, which can help distinguish it from normal up arrow notation
Finally, if needed, PAN can be written with \(\uparrow_P\) for best legibility. \begin{eqnarray*} a \uparrow^n b = \left\{ \begin{array}{ll} {a}^b & (n = 1) \\ a & (n > 1, b = 1) \\ (a \uparrow^{n-1})^{b\uparrow^{n-1}b} (a \uparrow^n (b-1)) & (n > 1, b > 1) \end{array} \right. \end{eqnarray*}

Definition of the third case

The third case referes to recursion.
\(f^{a+1}(x) = f^{a}(f^{a}(x))\), for \(a>0\)
\(f^{0}(x) = f(x)\) Termination case.
Note this is slightly different from normal recurstion, as the total iterations should be \(2^{a}\) In the above third case, we take \(f(x)\) to be \(a\uparrow^{n-1}(x)\). \(a \rightarrow b = a^{b}\)
